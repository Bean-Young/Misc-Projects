# Miscellaneous Projects

This repository was created by [***Yuezhe Yang***](https://bean-young.github.io) to archive small projects, course designs, experimental code, reports, and supporting materials.

## Overview

The repository contains coursework and project materials from programming, algorithms, computer vision, machine learning, signal processing, optimization, control, and related topics. It also includes standalone project archives such as DeepPET and Cat vs. Dog.

## Repository Map

| Area | Contents |
| --- | --- |
| `PET-Project/` | DeepPET project page materials, demo image, certificates, reports, and citation information. |
| `Computer Vision/` | Computer vision coursework and YOLOv3 project code. |
| `Python Programming/` | Python class exercises, coursework, Docker environment, and reports. |
| `Machine Learning/`, `Neural Network and Deep Learning/`, `Natural Language Processing/` | ML, deep learning, and NLP assignments and experiments. |
| `Data Structures and Algorithms/`, `Programming and Algorithm Comprehensive Practice/` | Algorithm practice code and reports. |
| `Digital Signal Processing/`, `Course Project of Signal and Information Processing/` | DSP and signal-processing coursework. |
| `Optimization method/`, `Principle of Intelligence Control/` | Optimization and control coursework. |
| Other course folders | Additional undergraduate coursework archives and related materials. |

## Usage Notes

This repository is an archive, so different folders may have different dependency requirements and coding styles. When using a project, start from the README inside that folder when available.

If you need to use the code or reports contained here, you may refer to them, but please ***do not copy*** them directly. If you find the content helpful, please consider giving this repository a star and using the citation below.

## Maintenance Notes

- Existing project files are preserved for archival completeness.
- Generated local files such as caches, editor metadata, and environment secrets are ignored by `.gitignore`.
- Some legacy course exercises are intentionally kept close to their original form, with only small compatibility or safety fixes applied.
